This storefront has the CFAnalytics tracker installed. Click anything on this page, browse to the shop, then open the dashboard — every page view and click shows up there as it happens.

Fast

Events batch in memory and flush on an interval, plus immediately on tab close via sendBeacon.

Accurate

Click coordinates are stored in pixels and as page-relative percentages for cross-device heatmaps.

Simple

One script tag, zero config. Works on any static or server-rendered page.
